The Wobbly Table Theorem

A square table on an uneven (but continuous) floor can always be stabilized by rotating it at most 90°.

How it works

Place a square table on a bumpy (but continuous) floor. Three legs always define a plane, but the fourth may hover or dig in. Define the wobble function:

f(θ) = (hA + hC) − (hB + hD)

where hi is the floor height at leg i's position when the table is rotated by angle θ. For a square table, rotating by 90° swaps the diagonal pairs (A,C) and (B,D), so:

f(θ + 90°) = −f(θ)

Since f is continuous (the floor is continuous) and changes sign, the Intermediate Value Theorem guarantees some θ* where f(θ*) = 0 — all four legs touch the ground.

Reference: Baritompa, Löwen, Polster & Ross, "Mathematical Table Turning Revisited" (2005). The full theorem requires a Lipschitz condition (floor slope ≤ ~35°) and sufficient leg length.
