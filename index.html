The Wobbly Table Theorem

A square table on an uneven (but continuous) floor can always be stabilized by rotating it at most 90°.

How it works

Place a square table on a bumpy (but continuous) floor. Three legs always define a plane, but the fourth may hover or dig in. Define the wobble function:

$$f(\theta) = \bigl(h_A(\theta) + h_C(\theta)\bigr) - \bigl(h_B(\theta) + h_D(\theta)\bigr)$$

where $h_i(\theta)$ is the floor height at leg $i$'s position when the table is rotated by angle $\theta$. For a square table, rotating by $90°$ swaps the diagonal pairs $(A,C)$ and $(B,D)$, so:

$$f(\theta + 90°) = -f(\theta)$$

Since $f(\theta)$ is continuous (the floor is continuous) and changes sign over $[0°, 90°]$, the Intermediate Value Theorem guarantees some $\theta^*$ where $f(\theta^*) = 0$ — all four legs touch the ground.

Limitations and generalizations

The theorem requires a square footprint — equal diagonals ensure that a 90° rotation swaps the diagonal pairs. For a rectangular table, this symmetry breaks and balance is not guaranteed. A tripod (three legs) never wobbles at all: three points always define a plane. The key insight is that the theorem applies to the geometry of the contact points, not the object itself — any four-footed object with a square base (like a chair) qualifies.

Reference: Baritompa, Löwen, Polster & Ross, "Mathematical Table Turning Revisited" (2005). The full theorem requires a Lipschitz condition (floor slope ≤ ~35°) and sufficient leg length.
