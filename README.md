# The Wobbly Table Theorem

An interactive 3D visualization demonstrating the mathematical theorem that guarantees you can always stabilize a wobbly four-legged table by rotating it.

## The Theorem

The Wobbly Table Theorem (also known as the intermediate value theorem application for tables) states that for any continuous, reasonably smooth floor surface, a square four-legged table can always be positioned so that all four legs touch the ground — simply by rotating the table around its center.

## Demo

**[Live demo](https://timvieira.github.io/table-theorem/)**

You can:

- Adjust the terrain bumpiness and shape
- Watch the table rotate and observe how leg contact changes
- See the theorem in action as the table finds a stable position

## How It Works

The visualization uses Three.js to render a 3D scene with a table on an uneven surface. As the table rotates, the simulation tracks which legs are in contact with the ground, illustrating the continuous rotation argument at the heart of the proof.
